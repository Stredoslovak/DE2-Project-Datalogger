bme280_settings
bme280 sensor settings structure which comprises of mode, oversampling and filter settings.
Definition bme280_defs.h:439

This API is used to calculate the maximum delay in microseconds required for the temperature/pressure/humidity(whichever are enabled) measurement to complete. The delay depends upon the number of sensors enabled and their oversampling configuration.

Parameters
[out]max_delay: Delay required in microseconds.
[in]settings: Contains the oversampling configurations.
Returns
Result of API execution status.
Return values
0-> Success.
>0 -> Warning.
<0 -> Fail.
